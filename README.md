# CBNU CCC Backend Server
&nbsp;이 레파지토리는 충북대학교 한국대학생선교회(CBNU CCC) 웹사이트의 백엔드 서버입니다. 자바 스프링부트를 이용하여 개발했습니다.

## 📃 Mainly Used Frameworks, Libraries or Services
* **Spring Boot 4**
* **Spring Security**
* **lombok**
* **slf4j**
* **Mailgun**<br>이메일 인증 등 이메일 전송용
* **Supabase**<br>데이터베이스
* **Railway**<br>서버 배포용
* **Docker**<br>Railway 컨테이너 제작용
